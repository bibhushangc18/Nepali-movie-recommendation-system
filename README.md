 About
This system recommends similar Nepali movies 
based on content features like genre and plot 
using Word2Vec and KNN algorithm.

 Technologies Used
- Python
- Pandas
- Scikit-learn (KNN)
- NLTK (Stemming)
- Gensim (Word2Vec)
- Streamlit (Web UI) — Coming Soon
- TMDB API (Movie Posters) — Coming Soon

Dataset
- 800 Nepali movies scraped from IMDb
- Features: Title, Genre, Plot, Rating, Year, Runtime

 Completed So Far
- Loaded and explored 800 Nepali movies
- Exploratory Data Analysis (EDA)
- Data Cleaning and Preprocessing
- Feature Engineering (Tags creation)
- Stemming using NLTK Porter Stemmer

 Coming Soon
- Word2Vec Vectorization
- KNN Model (K=5)
- Evaluation Metrics (Precision, Recall, F1)
- Streamlit Web App
- Movie Posters via TMDB API
- Deployment on Streamlit Cloud

 How it Works
1. Collect 800 Nepali movies from IMDb
2. Clean and preprocess data
3. Create tags from genre and plot
4. Stem words to root form
5. Convert text to vectors (Word2Vec)
6. Find similar movies using KNN
7. Return Top 5 recommendations
